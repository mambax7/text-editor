Insertion helper for special characters.

Start

Include the script just after the plugin instances, then set a new configuration data called emoji as true:

<script src="../text-editor.min.js"></script>
<script src="../text-editor/ui/ui.min.js"></script>
<script>
var editor = TE.ui(document.querySelector('textarea'), {
    symbol: true
});
</script>
<script src="../text-editor/ui/ui/special-characters/specal-characters.min.js"></script>

Configure

Symbol

Default symbol data are stored in editor.config.symbol:

// console.log(editor.config.symbol);

[

    // recent symbol
    [
        ['Recently Used', ""],
        []
    ],

    // all symbol
    [
        ['All Symbols', '❖'],
        [
            ['&quot;', 'QUOTATION MARK'],
            ['&amp;', 'AMPERSAND'],
            [ … ],
            …
            …
        ]
    ]
]

Each symbol classified in a particular category:

editor.config.symbol[1]; // All Symbols

Each category consists of a title and data:

editor.config.symbol[1] = [
    ['All Symbols', '❖'],
    [ … ]
];

Each category title consists of a title and its icon (optional):

editor.config.symbol[1][0] = ['All Symbols', '❖'];

Each category data consists of an icon, title and its custom string to insert (optional). By default, it will insert the first array but if the third array is defined, then it will insert the third array item:

editor.config.symbol[1][1] = [
    ['&quot;', 'QUOTATION MARK'],
    ['&amp;', 'AMPERSAND'],
    [ … ],
    …
    …
];

Replace

Replace the first symbol data (QUOTATION MARK) with a custom one:

// this will insert `❄`
editor.config.symbol[1][1][0] = ['❄', 'SNOW'];
// this will insert `*`
editor.config.symbol[1][1][0] = ['❄', 'SNOW', '*'];

Example with image icon:

// this will insert `<img src="../img/snow.png">`
editor.config.symbol[1][1][0] = ['<img src="../img/snow.png">', 'SNOW'];
// this will insert `*`
editor.config.symbol[1][1][0] = ['<img src="../img/snow.png">', 'SNOW', '*'];

Remove " symbol from “All Symbols” set:

editor.config.symbol[1][1][0] = false;

Custom Category

Add your own custom category set:

editor.config.symbol.push([
    ['My Symbol Set', '❤'],
    [
        ['A', 'LETTER A'],
        ['B', 'LETTER B'],
        [ … ],
        …
        …
    ]
]);

After doing any of the customizations above, you can call editor.update() to refresh the results.
