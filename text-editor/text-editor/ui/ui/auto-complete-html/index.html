Auto–close HTML tags and auto–add HTML attributes.

Start

Include the script just after the plugin instances, then set a new configuration data called auto_close_html as true:

<script src="../text-editor.min.js"></script>
<script src="../text-editor/ui/ui.min.js"></script>
<script>
var editor = TE.ui(document.querySelector('textarea'), {
    auto_close_html: true
});
</script>
<script src="../text-editor/ui/ui/auto-complete-html/auto-complete-html.min.js"></script>

Keyboard Hints

Press ␣ key after typing the left characters to produce the right characters:

  • <div|> → <div |></div>
  • <span|> → <span |></span>
  • <a|> → <a href="|"></a>
  • <img|> → <img alt="" src="|">
  • <input|> → <input type="|">

Press > key after typing the left characters to produce the right characters:

  • <div| → <div>|</div>
  • <span| → <div>|</span>
  • <img| → <img>| or <img />| if editor.config.suffix is />

Press d key after typing the left characters to produce the right characters:

  • <!|> → <!DOCTYPE html|>

Press - key after typing the left characters to produce the right characters:

  • <!|> → <!-- | -->
