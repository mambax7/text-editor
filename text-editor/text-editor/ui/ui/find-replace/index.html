Keyboard Hints

  • Press ⌘+F to Find and Replace.
  • Type something in the find field then press ↵ to find.
  • Press ⇥ after typing something in the find field. Type something in the replace field then press ↵ to replace.
  • Press ⌥ either in find field or replace field to change the search mode:
    • Ignore Case → a will match a and A
    • Match Case → a will match a but A
    • Match Pattern → [a-z]+ will match a, b, A, B, ab, aB, Ab and AB
    • Match Case and Pattern → [a-z]+ will match a, b and ab but A, B, aB, Ab and AB
